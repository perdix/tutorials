The Internet

The internet is a network of computer networks. It is the technical infrastructure for further applications.

The World Wide Web

The World Wide Web is a collection of technologies and protocols. Originally, it was used to publish scientific texts.
Today, it is a platform for exchange, interaction and multimedia.

Memorize

  • HTML Hypertext Markup Language - Eine Auszeichnungssprache, um eine Website zu gestalten.
  • HTTP(S) Hypertext Transfer Protocol (Secure). Ein technisches Protokoll zum Transport von Daten im Web. Wenn HTTPS verwendet wird, wird der Request und die Response verschlüsselt.
  • Client Ein Computer des Anwenders (Laptop, Smartphon, Stand-PC).
  • Server Hypertext Markup Language - Eine Auszeichnungssprache, um eine Website zu gestalten.
  • Webbrowser Eine Software, um Websites abzurufen. Z.B.: Chrome, Firefox, Opera, Safari, Edge.
  • Domain Einzigartige Adresse, an der der Server für alle Clients erreichbar ist
